Seaborn
    Seaborn is a Python data visualization library based on matplotlib.

Matplotlib -> Low level
Seaborn -> High level

Benifits of using Seaborn
    In-build datasets
    We get lot of high level features
    It has great support for the pandas dataframes (df)
    boxplots, heatmaps
