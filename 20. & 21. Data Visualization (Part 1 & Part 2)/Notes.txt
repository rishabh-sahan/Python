Pie Charts
A pie chart is a circular chart divided into slices to show relative proportions of a whole.
    Each slice = a category
    Size of slice = value of that category relative to total
Prefer them only for few categories: (max of 5-6 categories)

Customizations
autopct="%1.1f%%" - shows percentage of each slice
colors - add custom colors
Wedgeprops - define (edgecolor, linewidth, linestyle, alpha, fill...)
explode - highlights a slice by pulling it out 
startangle - rotate the chart 
shadow - adds 3D effect or makes the chart circular

Histograms
A histogram shows the distribution of numerical data by:
    Dividing values into bins (ranges)
    Counting how many values fall into each bin (frequency)

Ex:
            70 - 100
    inclusives - exclusives
    left value - right value

bins - continuous range

Histograms are used for the numerical data rather than the categorical data

Box Plots
    A box plot is a statistical visualization that summarizes the distribution of a dataset using five key numbers:
Minimum (lowest non-outlier)
First Quartile (Q1) - 25th percentile
Median (Q2) - 50th percentile
Third Quartile (Q3) - 75th percentile
Maximum (highest non-outlier)

Ex data: 7, 8, 5, 6, 9, 4, 10, 12, 15
Sorted Data: 4, 5, 6, 7, 8, 9, 10, 12, 15

    Q2 (Median) = 8
    Q1 (First Quartile) = 6
        4, 5, 6, 7, 8 -> Middle Value - 6
    Q3 (Third Quartile) = 10
        8, 9, 10, 12, 15 -> Middle Value - 10

Box Plots is also called as Box & Whiskerplot
Components: Whiskers & Outliers

    IQR - Inter-Quartile Range
        IQR = Q3 - Q1
    Lower Whisker = Q1 - (1.5*IQR)
    Upper Whisker = Q3 + (1.5*IQR)

Stack PLots
    A stack plot is a type of plot where multiple data series are stacked on top of each other.
    Each "layer" shows th contribution of one category, and the top line shows the cumulative total.
