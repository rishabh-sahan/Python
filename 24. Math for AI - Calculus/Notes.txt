Calculus
It is the branch of mathematics that studies continuous change.

Functions
    Each input has one & only output.
    Domain - all valid input values a function can take (x)
    Range - all possible output values the function can produce (y)

Nested Functions or Composite Functions

How functions behave?
    Scalar multiplication with x
    Scalar addition with x

Differentation
    Differentation describes how fast one quantity changes compared to another.
    So, how fast y changes when x changes = dy/dx
Differentation Rules
    Constant Rule
    d/dx(c) = 0

    Power Rule
    d/dx(x^n) = nx^n-1

    Product Rule

    Chain Rule
    dy/dx = dy/du * du/dx

Finding Minima/Maxima
    Minima - the lowest value of a function
    Maxima - the highest value of a function
